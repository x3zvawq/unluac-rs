# Generate

> **职责**：把 Readability 后的稳定 AST 与 `NameMap` 渲染成最终 Lua 源码文本。纯输出层。
>
> **不负责**：结构恢复、命名策略、可读性 sugar、补缺失语法节点。
>
> **例子**：`AstIf { cond, then_block } + NameMap` → `"if cond then\n  ...\nend"`

## 入口

```
src/generate.rs
  generate_chunk(ast, names, target, comment_metadata, options) -> Result<GeneratedChunk, GenerateError>
```

## 模块布局

```
src/generate/
  common.rs     GeneratedChunk / GenerateOptions / GenerateMode / QuoteStyle / TableStyle
               GenerateCommentMetadata / GenerateChunkCommentMetadata / GenerateFunctionCommentMetadata
  error.rs      GenerateError
  doc.rs        Doc IR（布局中间层）
  render.rs     Doc → String（行宽折行 / 缩进 / 着色）
  emit/
    mod.rs      generate_chunk 骨架
    expr.rs     AstExpr → Doc（含运算符优先级与括号规则）
    stmt.rs     AstStmt → Doc
    syntax.rs   标点 / 关键字 / 通用语法片段（含 long-bracket 字符串规则）
    names.rs    读取 NameMap
  debug.rs      dump_generate
```

## 数据流

```
AstModule + NameMap + GenerateOptions
  └─ generate_chunk
       ├─ emit/stmt.rs + emit/expr.rs   AST → Doc
       └─ render.rs                     Doc → String
            └─ GeneratedChunk { source: String, warnings: Vec<Warning> }
```

## 应优先复用的共享设施

| 设施 | 用途 |
|---|---|
| `emit/names.rs` | 读取 `NameMap`，获取当前函数上下文的 binding 名字 |
| `emit/syntax.rs` | 标点 / 关键字 / long-bracket 字符串选择（`[[...]]` vs `[=[...]=]`）/ numeric-for 步长省略 |
| `doc.rs` | 布局 IR，布局策略通过 Doc 而非直接拼字符串 |

## 维护规范

1. **Generate 不补前层丢失信息**：AST 中没有的节点（`global *` / `global<const>` / `SingleValue` / `local <close>`），Generate 不自行发明。
2. **优先级由 Generate 统一维护**：运算符优先级与括号规则集中在 `emit/expr.rs`，不让调用方依赖 AST "刚好安全"。
3. **布局与语义分开**：语义选择（emit）与行宽折行 / 缩进（Doc / render）不混在同一个函数里。
4. **`NameMap` 始终按函数上下文读取**：进入嵌套函数时切换到正确上下文，不用模块级平铺表。
5. **共享输出规则落到 `emit/syntax.rs`**：如字符串 long-bracket 选择、numeric-for 步长省略等，统一在此，debug / generate 收敛到同一套文本约定。

## 向后提供的事实

`GeneratedChunk`（含 `source: String` 最终源码文本）。

## 排错指引

| 症状 | 检查点 |
|---|---|
| 括号多余或缺失 | `emit/expr.rs` 运算符优先级 |
| 字符串输出格式错误 | `emit/syntax.rs` long-bracket / 转义规则 |
| 函数内变量名错误（名字来自外层） | `emit/names.rs` 函数上下文切换 |
| numeric-for 步长不该出现但出现了 | `emit/syntax.rs` 默认步长省略判断 |
| 想看 Generate 结果 | `unluac --debug --target-stage generate <file>` |
